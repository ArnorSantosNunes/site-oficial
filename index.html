Calculadora

O projeto de calculadora é uma aplicação web interativa que permite ao usuário realizar operações matemáticas básicas, como adição, subtração, multiplicação e divisão. Além dessas funções, a calculadora oferece suporte a números decimais e armazena o histórico de cálculos. O objetivo é proporcionar uma ferramenta prática e eficiente para atender às necessidades de diferentes usuários.
